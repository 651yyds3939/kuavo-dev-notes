# kuavo机器人上下位机通信与网络配置笔记

### 1. 网络基础环境
* **下位机 (NUC) IP**: `192.168.26.11` (默认)
* **上位机 (Jetson) IP**: `192.168.26.12` (默认)
* **连接方式**: 内部千兆网线直连

### 2. 远程操作常用指令
* **SSH 登录上位机**: `ssh leju_kuavo@192.168.26.12` (密码: leju_kuavo)
* **文件传输 (SCP)**: `scp -r [本地路径] leju_kuavo@192.168.26.12:[远程路径]`

### 3. 局域网代理共享 (蹭网方案)
* **场景**: 上位机需要下载 YOLO 模型或调用大模型 API。
* **步骤**:
    1. 下位机翻墙软件开启 `Allow LAN`。
    2. 上位机终端执行:
       ```bash
       export http_proxy="http://192.168.26.11:7890"
       export https_proxy="http://192.168.26.11:7890"
       ```

### 4. 常见排故 (Troubleshooting)
* **权限问题**: 实机运行必须 `sudo su` 获取 root 权限。
* **双机时间同步**: 如果 ROS 话题延迟大，检查两台电脑系统时间是否一致。
