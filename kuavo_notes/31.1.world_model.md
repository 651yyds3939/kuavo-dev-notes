## 第 1 章：一次性 / 每次开训前环境注入

工作目录 **`~/kuavo_all/leju_robot_wm`**，Conda 环境 **`isaaclab_wm`**。

```bash
# 1. 强杀残留 Kit / Python，清缓存锁
pkill -9 -f kit
pkill -9 -f python
rm -rf ~/.local/share/ov/pkg/isaac-sim-4.2.0/kit/cache/*

# 2. Conda + 工作空间
conda activate isaaclab_wm
cd ~/kuavo_all/leju_robot_wm
unset DISPLAY

# 3. Isaac Sim / Isaac Lab 环境变量
source ~/.local/share/ov/pkg/isaac-sim-4.2.0/setup_python_env.sh
export EXP_PATH=$HOME/IsaacLab/source/apps/isaaclab.python.headless.kit
export CARB_APP_PATH=$HOME/.local/share/ov/pkg/isaac-sim-4.2.0/kit
export ISAAC_PATH=$HOME/.local/share/ov/pkg/isaac-sim-4.2.0
export VK_ICD_FILENAMES=/usr/share/vulkan/icd.d/nvidia_icd.json
export WANDB_MODE=offline
```

**首次克隆本仓后额外执行一次**：

```bash
cd ~/kuavo_all/leju_robot_wm
python -m pip install -e exts/ext_template
```

* **Zsh `zoxide` 截断**：多行粘贴失败时先 `bash`，再执行后续单行 `&&` 命令。
* **Headless 下 `GLFW initialization failed`**：可忽略，不是训练失败。

### 5.1 S42/S46 舞蹈世界模型（推荐先跑通）


### 5.1.1 S42 奖励修复（upright 门控，2026-06 起必用）

旧 run（无 upright 门控）`termination_rate` 长期 ~0.78，Play 约 1–2s 即倒。**改奖励后必须从头训练**，不要 `--resume` 旧 checkpoint。

`punch_env_cfg.py` 已对齐 S49：`track_punch_*_upright_exp`（`min_upright=0.75`）、`flat_orientation_l2=-8`、`base_height=-4`。

改代码后重装扩展：

```bash
cd ~/kuavo_all/leju_robot_wm
conda activate isaaclab_wm
python -m pip install -e exts/ext_template
```

**从头训练（推荐 run 名 `wm_upright_v2`）**：

```bash
python scripts/tdmpc2/train.py \
  --task Legged-Isaac-Velocity-Flat-Kuavo-S42-Punch-TDMPC2-v0 \
  --headless \
  --run_name wm_upright_v2 \
  --wm_num_envs 512 \
  --max_iterations 8000
```

验收：`termination_rate` 降到 **0.3 以下** 再 Play；目标 `steps` > 500。

**标准 5000 iter（8GB 显存友好，`wm_num_envs=512`）**：

```bash
cd ~/kuavo_all/leju_robot_wm
conda activate isaaclab_wm && unset DISPLAY
source ~/.local/share/ov/pkg/isaac-sim-4.2.0/setup_python_env.sh
export EXP_PATH=$HOME/IsaacLab/source/apps/isaaclab.python.headless.kit
export CARB_APP_PATH=$HOME/.local/share/ov/pkg/isaac-sim-4.2.0/kit
export ISAAC_PATH=$HOME/.local/share/ov/pkg/isaac-sim-4.2.0
export WANDB_MODE=offline

python scripts/tdmpc2/train.py \
  --task Legged-Isaac-Velocity-Flat-Kuavo-S42-Punch-TDMPC2-v0 \
  --headless \
  --run_name wm_2048 \
  --wm_num_envs 512 \
  --max_iterations 5000
```

**冒烟 / 首次验证（少 iter、快出 checkpoint）**：

```bash
python scripts/tdmpc2/train.py \
  --task Legged-Isaac-Velocity-Flat-Kuavo-S42-Punch-TDMPC2-v0 \
  --headless \
  --run_name wm_first \
  --wm_num_envs 256 \
  --max_iterations 200 \
  --seed_steps 2000
```

**显存更大时提高并行 env（OOM 则退回 512）**：

```bash
python scripts/tdmpc2/train.py \
  --task Legged-Isaac-Velocity-Flat-Kuavo-S42-Punch-TDMPC2-v0 \
  --headless \
  --run_name wm_1024 \
  --wm_num_envs 1024 \
  --max_iterations 5000
```

* **默认行为**：未指定 `--wm_num_envs` 时，`train.py` 将 `num_envs` 上限钳为 **512**（8GB 笔记本友好）。
* **采集阶段**：`seed_steps=5000` 内用随机动作填 buffer；满 buffer 后才开始世界模型更新。
* **checkpoint**：每 **200** iter 存 `model_200.pt`、`model_400.pt` …；结束另有 `model_final.pt`。
* **训练时不开 MPPI**：`mpc=false`（仅策略先验采集）；Play 时默认 `mpc=true`。

## 第 6 章：断点续训（Resume）

`--resume` **只恢复模型权重**（`model_*.pt`），**不恢复** replay buffer、`global_step`、TensorBoard 步数；会新建带时间戳的 log 目录。

**示例：从 `model_3000.pt` 再训 2000 iter（凑满原计划 5000）**：

```bash
cd ~/kuavo_all/leju_robot_wm

export RUN_DIR=2026-06-22_05-12-54_wm_2048   # 替换为你的 run 文件夹名

python scripts/tdmpc2/train.py \
  --task Legged-Isaac-Velocity-Flat-Kuavo-S42-Punch-TDMPC2-v0 \
  --headless \
  --run_name wm_2048_resume \
  --wm_num_envs 512 \
  --resume \
  --load_run ${RUN_DIR} \
  --checkpoint model_3000.pt \
  --max_iterations 2000
```

* **`--load_run`**：相对于 `logs/tdmpc2/Kuavo/s42/tdmpc2_dance/` 的子目录名（不含 experiment 前缀）。
* **`--checkpoint`**：run 目录内的文件名，如 `model_3000.pt`、`model_final.pt`。
* 续训后 Play 请用 **新 run 目录** 里最新的 `model_*.pt`。

## 第 7 章：Play 可视化与录视频

```bash
cd ~/kuavo_all/leju_robot_wm
conda activate isaaclab_wm
unset DISPLAY
source ~/.local/share/ov/pkg/isaac-sim-4.2.0/setup_python_env.sh
export EXP_PATH=$HOME/IsaacLab/source/apps/isaaclab.python.headless.kit
export CARB_APP_PATH=$HOME/.local/share/ov/pkg/isaac-sim-4.2.0/kit
export ISAAC_PATH=$HOME/.local/share/ov/pkg/isaac-sim-4.2.0

export RUN_DIR=2026-06-22_05-12-54_wm_2048
export CKPT=model_3000.pt

# 带 GUI 看仿真（去掉 --headless）
python scripts/tdmpc2/play.py \
  --task Legged-Isaac-Velocity-Flat-Kuavo-S42-Punch-TDMPC2-Play-v0 \
  --checkpoint logs/tdmpc2/Kuavo/s42/tdmpc2_dance/${RUN_DIR}/${CKPT} \
  --eval_steps 2000

# Headless 录视频（约 200 步 ≈ 4s，50Hz）
python scripts/tdmpc2/play.py \
  --task Legged-Isaac-Velocity-Flat-Kuavo-S42-Punch-TDMPC2-Play-v0 \
  --headless \
  --checkpoint logs/tdmpc2/Kuavo/s42/tdmpc2_dance/${RUN_DIR}/${CKPT} \
  --video \
  --video_length 200

# 仅策略先验、不用 MPPI（对照实验）
python scripts/tdmpc2/play.py \
  --task Legged-Isaac-Velocity-Flat-Kuavo-S42-Punch-TDMPC2-Play-v0 \
  --headless \
  --checkpoint logs/tdmpc2/Kuavo/s42/tdmpc2_dance/${RUN_DIR}/${CKPT} \
  --no_mpc \
  --video \
  --video_length 200
```

* **视频路径**：`<run_dir>/videos/play/rl-video-step-0.mp4`
* **摔倒即停**：`base_contact` 触发后 `play.py` 会 `break`；终端打印 `Eval metrics: {'mean_reward': ..., 'steps': N}`，`steps` 很小（如 ~100）表示很快摔倒。
* **MPPI 限制**：`play.py` 在 `mpc=true` 时会把 `num_envs` 设为 **1**。

S49 Play：

```bash
python scripts/tdmpc2/play.py \
  --task Legged-Isaac-Velocity-Flat-Kuavo-S49-Punch-TDMPC2-Play-v0 \
  --headless \
  --checkpoint logs/tdmpc2/Kuavo/s49/tdmpc2_dance/<RUN_DIR>/model_3000.pt \
  --video \
  --video_length 200
```

## 第 9 章：默认超参与显存建议

来源：`exts/.../config/s42/agents/tdmpc2_cfg.py`（S49 见 `s49/agents/tdmpc2_cfg.py`）。

| 参数 | S42 舞蹈默认 | CLI 覆盖 |
|------|-------------|----------|
| `max_iterations` | 5000 | `--max_iterations` |
| `save_interval` | 200 | 无 CLI，改 cfg |
| `seed_steps` | 5000 | `--seed_steps` |
| `steps_per_env` | 24 | 无 CLI |
| `wm_num_envs` | 训练时默认钳 512 | `--wm_num_envs` |
| `action_scale` | 0.30 | 无 CLI |
| `batch_size` | 128（S49: 96） | 无 CLI |
| `buffer_size` | 300000（S49: 200000） | 无 CLI |
| `latent_dim` | 512 | 无 CLI（改 cfg） |
| `obs_profile` | `dance`（115 维） | 无 CLI |
| `mpc`（训练采集） | false | `--mpc`（极慢，一般不开） |

**显存粗估（S42 舞蹈 + 512 envs）**：约 **6–8 GB**；OOM 顺序下调：`--wm_num_envs 512 → 256 → 128`。

**舞蹈 CSV 帧数**：`kuavo_action_FULLBODY_INPLACE_RAD.csv` 约 **1565** 帧，与世界模型 `episode_length=1565` 对齐。

## 第 10 章：附录排障

| 现象 | 处理 |
|------|------|
| Kit 黑屏 / 卡加载 | 第 1 章 `pkill` + 清 `kit/cache` |
| CUDA OOM | 降低 `--wm_num_envs` |
| `train/termination_rate` 长期 >0.7 | 正常中期现象；跑满 iter 或续训；Play 仍倒则换后期 `model_*.pt` |
| Play 视频只有 ~2s、往一侧倒 | `steps` 很小 = `base_contact` 摔倒终止，非视频截断问题 |
| Resume 后曲线从 0 开始 | 预期行为；只看新 run 的 `tb/` |
| S49 训练 NaN / 站不稳 | 重跑 `setup_s49_training_assets.sh` + `rm -rf /tmp/IsaacLab/usd_*` |
| 找不到 CSV | 确认 `leju_robot_wm` 根目录有 `kuavo_action_FULLBODY_INPLACE_RAD.csv` 等文件 |
| `which python` 不对 | 重新 `conda activate isaaclab_wm` |
