## 第 4 章：MCP HTTP 桥（`mcp/mcp_http_bridge.py`）— 生产路径

> NUC 系统 Python **3.8.10**，**装不了 fastmcp**（需 ≥ 3.10）。  
> 因此 **2026-06 实机联调全部走 HTTP 桥**，`mcp_kuavo_server.py` 保留作实验对照。

### 4.1 工具与底层映射

| HTTP Tool / Resource | 底层函数（`moveit_auto_grasp`） | 说明 |
|----------------------|--------------------------------|------|
| `locate_target_yolo(object_name)` | `_collect_vision_targets_tf2_style()` | 10 帧 TF2 采点 + `np.median`；`raw_y>0` 判左手 |
| `calculate_ik_trajectory(x,y,z,is_left)` | `_solve_pose_ik()` + `tcp_offsets_for_arm` | MoveIt `/compute_ik`；**入参 x,y 为 raw，桥内再加 TCP** |
| `execute_arm_motion_batch(joint_chain, total_time, is_left, close_claw)` | `execute_single_pose` / `publish_arm_trajectory_batch` | 泵送至 `/kuavo_arm_target_poses` |

| `tts_speak(text)` | HTTP POST | `http://192.168.26.1:5000/tts` |

| `GET /resource/robot/telemetry/status` | `joint_states_callback` 缓存 | 14 轴位置 + 末次下发 + arm mode |

### 4.5 大模型推荐 Tool 调用链

```text
1. locate_target_yolo("水瓶")          → 得 raw_x, raw_y, is_left_arm
2. calculate_ik_trajectory(raw_x, raw_y, z, is_left)   ← 预瞄 / 抓握 / 抬升各调一次
3. execute_arm_motion_batch([[...14 rad...]], total_time, is_left=is_left)
4. execute_arm_motion_batch(..., close_claw=True)      ← 抓握步，走 claw_safe
5. emergency_joint_space_override(is_left)             ← 收手（test_mcp_grasp 步骤 6）
6. 若 IK 失败 → emergency_joint_space_override(is_left)
7. tts_speak("抓取完成") 或 tts_speak("逆解失败，已安全撤离")
```

### 4.6 ⚠️ IK 入参：禁止「双重 TCP」

NUC 的 `calculate_ik_trajectory` 会对 `x,y` **在桥内再加一次** TCP 偏置（`tcp_offsets_for_arm`）。

| 传给 IK 的坐标 | 正确？ |
|----------------|:------:|
| `locate_target_yolo` 返回的 **`raw_x / raw_y`** | ✅ |
| `locked_x / locked_y`（已加过 TCP） | ❌ 会偏一倍 |

**预瞄点**在 Orin 侧按 **locked 坐标**（已加 TCP）计算后，传给 IK 时需反算：

```text
ik_x = pre_x - tcp_offset_x
ik_y = pre_y - tcp_offset_y
```

**抬升点**：`x,y` 用 raw，`z = 0.605`（0.385 + 0.22）。

### 4.7 左右手选臂规则

与 22.2 行为树、`moveit_auto_grasp.py` 完全一致：

| 条件 | 选用 |
|------|------|
| `raw_y > 0` | 左手（`is_left=true`） |
| `raw_y < 0` | 右手（`is_left=false`） |

`locate_target_yolo` 自动按 `raw_y` 正负返回 `is_left_arm`。  
Orin `test_mcp_grasp.py` 只有 **`--is-left`** 强制左手；**没有 `--is-right`**，右手 = 不加 `--is-left`。

## 第 6 章：LeRobot 数据收割机（`mcp/lerobot_data_harvester.py`）

> **状态：代码已写好，实机 100 ep 采集待启用。** LeRobot 先不管，MCP 抓取链路跑通后再开。

### 6.1 功能概述

- **专家工人**：直接调用 `moveit_auto_grasp._run_grasp_sequence()` 全闭环管线。
- **Episode 数**：默认 100 次（`--episodes 100`）。
- **采样频率**：50Hz（每 20ms 一次）。
- **录制字段**：

| 字段 | 来源 | 单位/格式 |
|------|------|----------|
| `observation.images.head` | `/camera/color/image_raw`（Orin 头部相机经 ROS 网传） | RGB，480×640×3 |
| `observation.state` | `mag.current_joints_rad` | 14 轴弧度 |
| `action` | `mag.last_commanded_joints_rad` | 14 轴目标弧度 |

- **输出格式**：LeRobot v3.0 — `LeRobotDataset.create` → `add_frame` → `save_episode` → `finalize()` → Parquet + MP4。

### 6.2 抓取生命周期（与 `moveit_auto_grasp` 一致）

```text
曲肘护胸 → 退至预瞄点（12cm 外）→ 水平插入抓握点
    → claw_safe 闭爪 → 垂直抬升 22cm → vla 收手（肩膀外摆 75°）
    → 曲肘护胸 → init 下垂 → 开爪释放
```

### 6.3 目标采点策略

| 模式 | 参数 | 行为 |
|------|------|------|
| 视觉采点（默认） | `--use-vision` | `_collect_vision_targets_tf2_style()` 收满 10 帧 |
| 视觉失败回退 | 自动 | 随机桌面空域坐标 |
| 纯随机 | `--no-vision` | 不依赖 YOLO，仅用于 IK 调试 |

**随机桌面空域（base_link 米）**：

| 轴 | 范围 |
|----|------|
| X | 0.38 ~ 0.52 |
| Y（左手） | 0.02 ~ 0.12 |
| Y（右手） | -0.12 ~ -0.02 |
